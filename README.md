# undefined.academy

A free, 16-week Full-stack JavaScript bootcamp platform. The site serves as both a landing page and enrollment hub for an intensive online program designed to take aspiring developers from the basics of the terminal and HTML/CSS all the way through advanced topics like Docker and CI/CD — at no cost to the student.

## Program Content

**16 weeks · Full-stack JavaScript · 100% free · ~10h/week**

### Week 1 — Terminal, Markdown, Git & HTML/CSS Fundamentals
- Dev environment setup (Ubuntu, Git, SSH, Node.js, VS Code)
- Markdown syntax
- Terminal basics & aliases
- Git: commits, branches, merge, rebase, remote repos
- HTML: page structure, hyperlinks, media
- CSS: box model, selectors, cascade, specificity, layout

### Week 2 — Programming, JavaScript 101, Semantic HTML & the DOM
- Introduction to programming concepts (algorithms, operators, types)
- JavaScript fundamentals: variables, equality, objects, functions, scope, loops
- Semantic HTML: document outline, sectioning elements, forms, controls
- JSON, DOM and CSSOM

### Week 3 — Flexbox, CSS Grid, Responsive Design & UI Frameworks
- Flexbox and CSS Grid in depth
- Positioning and responsive design
- History of JS bundling: Webpack and Vite
- Bootstrap 5 and Tailwind CSS

### Week 4 — JS Libraries, SASS & Design Tokens
- Lodash and Day.js (and when to skip them)
- SASS and CSS architecture evolution
- Design tokens with CSS custom properties

### Week 5 — ES2015+, Async Programming & Advanced JavaScript
- Modern JS: template literals, destructuring, arrow functions, optional chaining
- Collections: Map, Set, WeakMap, WeakSet
- Array, Object, String and Math methods
- Async JS: callbacks, promises, generators, async/await
- Advanced topics: prototypal inheritance, closures, `this`, functional programming, regex

### Week 6 — JS in Production, Intro to React & Hooks
- Production tooling: Babel, minification, source maps, code splitting
- React fundamentals: JSX, props, state, lifecycle
- Events and forms in React
- Higher-order components, render props, hooks

### Week 7 — Redux & Building Apps with React / Next.js
- Redux: actions, reducers, store, data flow
- Async actions and middleware
- React Redux and `useRedux`
- Building dashboards with Create React App and Next.js

### Week 8 — Node.js, Express.js & MongoDB
- Node.js vs browser JavaScript
- Express.js: server setup and middleware
- MongoDB: connecting from Node.js, Mongoose

### Week 9 — SQL Databases & PostgreSQL
- Entity-relationship diagrams
- Connecting to PostgreSQL from Node.js
- Essential SQL queries

### Week 10 — Building REST APIs
- REST APIs with Express.js, Hapi.js and Sails.js
- Firebase as an API backend

### Week 11 — Authentication, OAuth 2.0 & Auth0
- Passport.js authentication
- How OAuth 2.0 works
- Auth0 for MVPs

### Week 12 — Web Security
- Injection attacks and how to prevent them
- Secure authentication patterns
- Avoiding sensitive data exposure
- Server permissions and configuration

### Week 13 — Servers: Linux, SSH, Digital Ocean & Vercel
- Linux file system and user permissions
- SSH into remote servers, VIM basics
- Deploying to Digital Ocean, Fly.io and Vercel

### Week 14 — Advanced CSS: Typography, Animations & Design Systems
- Typography in CSS
- Transitions and animations
- Building design systems with CSS

### Week 15 — Testing: Unit, Snapshot, Mocking & E2E
- Testing principles
- Unit and snapshot testing (React)
- Mocking strategies
- Integration and end-to-end tests

### Week 16 — Docker, Kubernetes & CI/CD
- Containers: Docker setup and configuration
- Kubernetes with Google Cloud Platform
- Continuous integration and continuous deployment pipelines
